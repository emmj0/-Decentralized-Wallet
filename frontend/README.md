# Frontend (React + Tailwind)

This folder will contain the React + Tailwind frontend for the project.

Scaffold notes:
- We'll use Vite + React for fast development.
- Tailwind CSS for styling.

To initialize locally (example using npm + Vite):

```powershell
cd frontend
npm install
npm run dev
```
